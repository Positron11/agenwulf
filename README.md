# Agenwulf Color Scheme

A GTKSourceView color scheme based on the Monokai theme, with support for the [Remarqu√©](https://github.com/Positron11/remarque) tree-based note-taking language. Dark and Light variants available.

## Installation & Usage

```bash
git clone https://github.com/Positron11/agenwulf-color-scheme.git
sudo mv agenwulf-color-scheme/GTKSourceView-5/agenwulf_*.xml /usr/share/gtksourceview-5/styles/
rm -rf ./agenwulf-color-scheme
```
